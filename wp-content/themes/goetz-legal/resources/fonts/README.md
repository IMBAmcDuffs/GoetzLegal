# Roboto webfont

`roboto-latin-300-700.woff2` is the unmodified Latin Roboto variable file
served by Google Fonts for the theme's original Chromium request at weights
300, 400, 500, and 700.

- Source: `https://fonts.gstatic.com/s/roboto/v51/KFO7CnqEu92Fr1ME7kSn66aGLdTylUAMa3yUBA.woff2`
- Stylesheet source: `https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;700&display=swap`
- Retrieved: 2026-07-18
- SHA-256: `1404ca348bd75ef836f4dd8b6f2cc719458642d1237c368296b2fc652dca47dc`
- License: SIL Open Font License 1.1; see `OFL.txt`.

The local copy removes the cross-origin stylesheet and font swap while keeping
the exact font file and discrete weight descriptors used by the approved site.
